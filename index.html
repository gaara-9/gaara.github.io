Introduction

Flutter is Google SDK (System Development Kit), which you can use to make Cross Platform Applications with a Single Code Base, i.e just one code and the Application can run on different platforms, like Android, iOS, Web, and some other platforms. Flutter is the best choice among different Cross Platform SDKs, because of the Native like Performance, and ease of making Beautiful, Smooth,and Native Looking Applications.

Dart

Flutter uses a Google self-built Powerful Programming language that is Dart, which is quite simple and easy to learn. If you have already worked on Java, Swift, or JavaScript, it will help you to learn Dart easily, as Dart is kind of a Combination of JavaScript and Java Goodness. For the language tour. Go to the Dart Language Tour Page.

Perquisite for starting Flutter Application Development:
You need the have below mentioned skills or tools in order to start Flutter application development.
1. Knowledge and Practice of Programming (Specially Object Oriented Language)
2. An IDE for development (Android Studio or VSCode)
3. Flutter SDK
